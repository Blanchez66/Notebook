??? tip "一些缩写"
    P: Price
    M: 总预算或收入

### 1.1 效用
**效用（U）：**消费者消费某种物品所得到的满足感。

在讨论一种物品的消费是否对消费者有效用时，经济学并不考虑其本身的价值判断问题。

!!! tip
    经济学在分析“某种消费是否给消费者带来效用（utility）”时，不会去评判这种消费“好不好、道不道德、值不值得”。

    它只关心：消费者自己有没有从中感到满足。只要消费者自己愿意花钱购买，并从中获得满足感，那么经济学就认为这是有效用的（positive utility）。

### 1.2 总效用与边际效用
**总效用（TU）：**消费者消费一定量某种物品所得到的总满足感。
    
    - 消费一种商品X，总效用函数为：$TU=U（X）$
    - 消费X、Y两种商品，总效用函数为：$ TU=U（X,Y）$

**边际效用（MU）：**消费者**增加一单位**某种物品消费所增加的总效用。

    - $MU_x = \frac{\partial TU}{\partial X}$
    - $MU_y = \frac{\partial TU}{\partial Y}$
    - 
!!! question
    + 为什么边际效用是递减的？
    + 边际效用递增意味着什么？ 
    ??? success "Answer"
        + 需要被满足之后，继续满足带来的快乐就下降了。
        + 你消费某种商品越多，从每一单位中获得的额外满足反而越来越高。❗边际效用递增非常罕见，通常只在特殊情境下出现。(收藏品、稀缺品)

### 1.3 边际效用递减规律
**边际效用递减规律：**在一定时间内，消费者消费某种物品的边际效用将随着消费量的增加而减少。

    - $\frac{dTU}{dX} > 0$
    - $\frac{d^2TU}{dX^2} < 0$

### 1.4 消费者均衡的条件
**消费者均衡**是指消费者在收入和商品价格既定的条件下作出实现效用最大化的消费选择。

**约束条件：** $M = P_x X + P_y Y$，（你花在 X 上的钱） + （你花在 Y 上的钱） = （你的总预算）。

**均衡的实现条件：** $\frac{MU_x}{P_x} = \frac{MU_y}{P_y} = \lambda$

!!! tip "消费者均衡条件的解释"

    - $\frac{MU_x}{P_x} > \frac{MU_y}{P_y}$：消费者将增加对X商品的购买，减少对Y商品的购买。
    - $\frac{MU_x}{P_x} < \frac{MU_y}{P_y}$：消费者将增加对Y商品的购买，减少对X商品的购买。

??? note "📈 消费者均衡的数学解"

    根据消费者的目标函数$U = U(X, Y)$和约束条件$M = P_x X + P_y Y$
    
    构造拉格朗日函数，
    $$L = U(X, Y) + \lambda(M - P_x X - P_y Y)$$

    分别对$X/Y/\lambda$求偏导，
    $$\frac{\partial L}{\partial X} = \frac{\partial U}{\partial X} - \lambda P_x = 0$$

    $$\frac{\partial L}{\partial Y} = \frac{\partial U}{\partial Y} - \lambda P_y = 0$$

    $$\frac{\partial L}{\partial \lambda} = M - P_x X - P_y Y = 0$$
    得到，
    $$\frac{MU_x}{P_x} = \frac{MU_y}{P_y} = \lambda$$
    $$M = P_x X + P_y Y$$

### 1.5 需求曲线的推导
- 消费者对某商品愿意支付价格的高低取决于消费该商品所得到的边际效用的大小。
- **边际效用递减规律**决定了需求曲线向**右下方**倾斜。

### 1.6 消费者剩余
**消费者剩余：**消费者**愿意支付**的价格总额**减去实际支付**的价格总额的差额，是消费者的一种主观心理感受。简单来说，就是你“感觉自己赚到了”的部分。

消费者剩余代表了消费者的福利。

### 2.1 偏好及其公理
**偏好：**消费者对某种商品的喜好或嗜好。

**偏好的三个公理：**

    - **完备性：**消费者能对不同的商品或商品的不同组合按照自身偏好排序。
    - **传递性：**偏好在逻辑上的一致性。
    - **非饱和性：**都是好商品，所以越多越好。
